# How to Deploy TaskVibe

TaskVibe is a **static web application**, meaning it runs directly in the browser without a backend server. You can deploy it for free using various services.

## Option 1: Netlify Drop (Easiest)
1.  Locate your `TaskVibe-Web.zip` file (or the folder containing `index.html`).
2.  Go to [Netlify Drop](https://app.netlify.com/drop).
3.  Drag and drop the zip file or folder onto the page.
4.  Your site will be live instantly!

## Option 2: GitHub Pages
1.  Create a new repository on GitHub.
2.  Push your project files (`index.html`, `style.css`, `app.js`, `elements/`).
3.  Go to Repository Settings -> Pages.
4.  Select the `main` branch as the source.
5.  Your site will be deployed at `https://<username>.github.io/<repo-name>`.

## Option 3: Vercel
1.  Install Vercel CLI (`npm i -g vercel`) or go to [vercel.com](https://vercel.com).
2.  Run `vercel` in the project folder.
3.  Follow the prompts.
